# dotfiles 

Automatically setup @HelloGrayson's Fedora Silverblue machine.

## How to run

No need to check this repo out, just open the terminal and run:

```console
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply HelloGrayson
```
